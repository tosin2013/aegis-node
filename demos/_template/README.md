# Demo template

Copy this directory and rename to start a new demo. The files:

- `manifest.yaml` — Permission Manifest (per ADR-004). MUST set
  `inference.determinism.seed = 42` and `temperature = 0.0` per
  ADR-020.
- `prompt.txt` — single-line user message that goes to
  `aegis run --prompt`. With deterministic sampling, the model's
  response is reproducible byte-for-byte across runs.
- `demo.tape` — [VHS](https://github.com/charmbracelet/vhs) recording
  script. Each `Type`/`Enter` line drives the terminal; `Sleep` lets
  the rendered GIF breathe at the right tempo.
- `demo.gif` — generated by `make`. Committed so CI can verify the
  SHA-256 hasn't drifted.
- `README.md` — what this demo demonstrates, which features (F1–F10) it
  exercises, and what the viewer should look for in the GIF.

## Render

From the demos directory:

```bash
make <NN>-<scenario-name>
```

Or render everything: `make all`.
